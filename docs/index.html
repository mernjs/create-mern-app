Getting Started

Getting started is easy!

MERN.JS is a super simple boilerplates and starter kits for building web and mobile apps in pure JavaScript, released as free and open-source software under the MIT License. MERN.JS Documentation

Current Version

v1.0.0

Installation

    npm install -g create-mernjs-app

    create-mernjs-app my_new_app
    
    cd my_new_app
    
    npm run dev

What is MERN.JS?

MERN.JS is a super simple boilerplates and starter kits for building web and mobile apps in pure JavaScript, released as free and open-source software under the MIT License.

Why is MERN.JS?

MernJs allows you to create a new MernJs app within seconds. It is officially maintained by the creators of MernJs
